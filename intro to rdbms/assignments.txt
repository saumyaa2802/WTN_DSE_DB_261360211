 Observe the structureas of a given EMPLOYEE table as below and suggest if this table can be further Normalized
 EMPNO,ENAME,SAL,DEPTNO,DNAME,LOC

All values are atomic (1NF satisfied).
The primary key is EMPNO (single attribute), so there are no partial dependencies.
